#  Sistema de Transportadora

Projeto desenvolvido como parte de um desafio proposto pelo **Geração Caldeira**, com o objetivo de praticar **lógica de programação** e **conceitos fundamentais de Java**, sem o uso de frameworks como Spring Boot.

O sistema simula o funcionamento básico de uma **transportadora**, permitindo o gerenciamento de encomendas, clientes e processos relacionados ao transporte.

---

##  Tecnologias Utilizadas

- **Java 21**
- Programação Orientada a Objetos (POO)
- Estruturas de controle (condicionais e laços)
- Coleções (List, Map, etc.)
- Entrada e saída de dados via terminal
- Organização em camadas simples (sem framework)

---

##  Objetivo do Projeto

Este projeto tem caráter **educacional**, sendo desenvolvido para:

- Exercitar a **lógica de programação**
- Consolidar conceitos aprendidos durante o curso
- Praticar **POO na prática**
- Desenvolver um sistema funcional **sem frameworks**
- Simular regras reais de um sistema de transportadora

---

##  Funcionalidades

O sistema permite:

-  Cadastro de viagens
-  Consulta de encomendas cadastradas
-  Cálculo de valores de transporte (ex: por peso ou distância)
-  Listagem geral de dados
-  Remoção ou atualização de registros

*(As funcionalidades podem variar conforme a evolução do projeto.)*

---

## Conceitos Aplicados

- Classes e objetos
- Encapsulamento
- Métodos e atributos
- Estruturas condicionais (`if`, `switch`)
- Estruturas de repetição (`for`, `while`)
- Manipulação de listas e dados
- Boas práticas de organização de código

---

## Observações

- Este projeto não utiliza Spring Boot ou qualquer framework.
- Toda a lógica foi desenvolvida manualmente para fins de aprendizado.
- O foco está na clareza do código e no entendimento dos conceitos.
